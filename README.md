# 🍷 Vinheria Agnello

Site institucional da **Vinheria Agnello**, desenvolvido como projeto acadêmico para apresentar a loja, seus produtos e história, com foco em design elegante e experiência do usuário.

---

## 📋 Sobre o Projeto

A Vinheria Agnello é uma loja fictícia especializada em vinhos nacionais e importados. O site foi criado para oferecer uma vitrine digital da vinheria, com seções de apresentação, catálogo de produtos, história da empresa e informações de contato.

---

## ✨ Funcionalidades

- **Verificação de idade** — Pop-up de confirmação de maioridade (18+) ao acessar o site, com redirecionamento para o Google caso o usuário seja menor de idade.
- **Catálogo de vinhos** — Exibição de produtos com imagem, nome, origem e uva.
- **Filtro lateral** — Sidebar com categorias e marcas para navegação no catálogo.
- **Seção "Nossa História"** — Apresentação da origem e valores da vinheria.
- **Rodapé com contato e redes sociais** — E-mail, telefone e links para Facebook, Instagram e Twitter/X.
- **Design responsivo** — Layout adaptado para dispositivos móveis (breakpoint em 768px).

## 🚀 Como Executar

1. Clone ou baixe o repositório:
   ```bash
   git clone https://github.com/UXcellence/Site-Vinheira_Agnello.git
   ```

2. Navegue até a pasta do projeto:
   ```bash
   cd Site-Vinheira_Agnello
   ```

3. Abra o arquivo `index.html` no seu navegador.

> Nenhuma dependência ou instalação adicional é necessária — o projeto é 100% front-end estático.
